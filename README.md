# GCP Infrastructure Deployment using Terraform

This project demonstrates Infrastructure as Code (IaC) by provisioning Google Cloud Platform (GCP) resources using Terraform and the HashiCorp Google provider.

---

## 🔧 Project Overview

- Configured Terraform Google provider
- Authenticated using gcloud CLI and Application Default Credentials
- Implemented environment-based variables (dev/test/prod)
- Initialized Terraform backend and provider plugins
- Executed `terraform init` and `terraform plan`
- Validated provider configuration and authentication

---

## 🏗 Technologies Used

- Google Cloud Platform (GCP)
- Terraform
- HashiCorp Google Provider
- gcloud CLI
- IAM
- Infrastructure as Code (IaC)

---

## 🚀 Deployment Workflow

1. Configure Google Cloud SDK (`gcloud init`)
2. Authenticate using Application Default Credentials
3. Configure Terraform provider block
4. Run `terraform init`
5. Run `terraform plan`
6. Validate infrastructure configuration
